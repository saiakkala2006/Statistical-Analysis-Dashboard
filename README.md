# 📊 Statistical Analysis & Data Mining Dashboard

An interactive web dashboard that performs **statistical analysis** and **data mining** on structured datasets using **PostgreSQL**, **Flask**, and **React**. The application transforms raw SQL queries into meaningful visual insights, enabling users to explore trends, detect anomalies, and analyze data efficiently.

---

## 🚀 Overview

This project demonstrates how SQL can be used beyond traditional CRUD operations by implementing advanced analytical techniques such as statistical computation, outlier detection, aggregation, trend analysis, and ranking through an interactive dashboard.

The application combines a PostgreSQL backend with a modern React frontend, allowing users to execute analytical operations and visualize results in real time.

---

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Statistical-Analysis-Dashboard.git
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Database

Create a PostgreSQL database and execute the SQL scripts located inside

```
database/
```

to create tables, functions, procedures, views, and sample data.
